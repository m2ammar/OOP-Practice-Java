# OOP Practice - Java

A collection of Object-Oriented Programming lab tasks and practice files written in Java, covering core OOP concepts as part of my BSCS coursework at KSBL (Karachi School of Business & Leadership).

## 📁 Structure

```
src/
├── OOPLab1/        - Introduction to Classes & Objects
├── OOPLab2/        - Constructors & Encapsulation
├── OOPLab4/        - Inheritance & Method Overriding
├── LabTask13march/ - Lab task (March 13)
└── ...             - Additional practice files
```

## 🧠 Concepts Covered

- **Encapsulation** — private, protected, public fields, getters/setters
- **Inheritance** — `extends`, `super` keyword, parent-child relationships
- **Polymorphism** — method overloading & overriding (`@Override`)
- **Abstract Classes** — partial abstraction, abstract methods
- **Interfaces** — full abstraction, `implements`
- **Packages** — organizing classes into packages
- **Constructors** — default, parameterized, constructor chaining
